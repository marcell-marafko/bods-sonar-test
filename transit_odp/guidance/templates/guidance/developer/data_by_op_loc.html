{% trans 'Getting data by operator' %}

{% blocktrans %} For all data types you can apply the NOC query parameter limiting results to datasets from operators that are associated with that code. {% endblocktrans %}

{% blocktrans %} The Bus Open Data Service allows operators to attribute National Operator Codes (NOCs) to their publishing accounts. When the NOC query parameter is applied to an API call, it will return all data sets for that NOC, and all other NOCs associated with that account. {% endblocktrans %}

{% blocktrans %} The full NOC database can be found on the {% endblocktrans %} {% filter force_escape %} {% trans 'Traveline website.' %} {% endfilter %}

{% trans 'Timetables' %}

{% blocktrans %} Use the query parameter adminArea by specifying a comma separated list of admin areas defined by their AtcoAreaCode as specified in the NPTG area codes. {% endblocktrans %}

{% trans 'Bus location' %}

{% blocktrans %} Use the query parameter boundingBox to limit results to bus location data with vehicle positions within the rectangular co-ordinates specified. {% endblocktrans %}

{% trans 'Fares' %}

{% blocktrans %} Use the query parameter boundingBox to limit results to fares data that contain information for the area within the rectangular boundingBox within the rectangular co-ordinates specified. {% endblocktrans %}

{% blocktrans %} Alternative, you could use the adminArea query paramter based on NPTG locality. {% endblocktrans %}
